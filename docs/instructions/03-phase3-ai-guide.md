# Phase 3 AI Guide

## Goal
Design clean system architecture.

## Architecture Decisions
- Service layer pattern
- Server/Client separation
- RLS policies
- Component hierarchy

## Folder Structure
- src/app/ (routes)
- src/components/ (UI)
- src/services/ (data)
- src/lib/ (utilities)

## Database Design
- Consistent naming (snake_case)
- Foreign key constraints
- Indexes on queries
- RLS on all tables
